## Page 1 · 钩子 + 问题

**页标题：我的视频没一样是我做的 —— 因为 80% 根本不是创作**

要点：
- 我发的每条视频：剪辑不是我剪的、封面不是我做的、文案不是我写的，全是 Agent 干的。我非技术背景
- 一条视频的真实流程：想选题 → 写稿 → 拍 → 剪 → 做封面 → 写三平台文案 → 挨个上传
- 真正创作的只有选题和拍摄，剩下全是重复体力活 —— 一个人做一条几小时。想日更，等于给自己开个没工资的血汗工厂

讲话稿：
> 先说个反常识的。我发的视频，剪辑不是我剪的、封面不是我做的、文案不是我写的，全是 Agent 干的，而我非技术背景。为什么这事重要：一条视频要写稿、剪辑、抠封面、给小红书抖音 X 各写一套文案、再挨个上传。真正算创作的只有选题和拍摄，剩下八成全是体力活，一个人做一条好几小时。想日更？等于给自己开了个不发工资的血汗工厂。

（视觉占位：三张我真实发布的成片截图并排 —— 剪辑成片 / 封面 / 三平台文案）

## Page 2 · 方案

**页标题：一条想法进来，三平台发布包出去**

要点：
- Creator OS：内容创作者的一人公司操作系统
- 7 个模块化 Agent，像流水线上的工位：想法 → 脚本 → 拍摄 → 剪辑 → 封面 → 文案 → 发布
- 每个工位可单独用，也可串成流水线
- 每个 Agent 的提示词外置可调 —— 模板谁都能用，调完就成了你自己的

讲话稿：
> Creator OS 是内容创作者的一人公司操作系统。7 个 Agent，像 7 个工位：口述想法进来，脚本 Agent 写成带钩子的口播稿，拍完登记，剪辑出成片，封面出图，文案一键出小红书抖音 X 三套，最后进发布台。每个工位能单独用，也能一路串下来。而且每个提示词都外置，你能自己调——调完它就懂你了。

（视觉占位：7 站流水线示意图，工位横向排列，每站一个图标）

## Page 4 · 差异化 + 真实数据

**页标题：越用越懂你账号的 AI —— 而且我自己天天在用**

要点：
- 别人做工具帮你干活，但永远不知道这条会不会火。我们的壁垒是校准闭环：
  1. 发布前，AI 盲预测表现，写进不可篡改的预测日志
  2. 发布第 3 天，自动抓真实数据和预测对账
  3. 差距写回评分标准 —— AI 越用越懂这个账号
- 每个创作者的校准数据是别人拿不走的独有资产（命令行版已跑通）
- 这不是 PPT，我就是第一个用户。实测：视频预处理 56 秒无人值守；一条人工动手约 10 分钟（此前全手工要几小时）；封面工厂 CoverLab 已上线，17 模板，cover-factory-web.vercel.app

讲话稿：
> 这是我们和所有 AI 工具最不一样的地方。别的工具帮你干活，但永远不知道这条会不会火。我们有个校准闭环：发布前 AI 先盲猜表现，写进改不了的日志；发第 3 天自动抓真实数据对账；差在哪写回评分标准。它越用越懂你这个号，这份数据是你独有的，换工具都带不走。而且这不是设想——我就是第一个用户。预处理 56 秒没人管；我人工动手大概 10 分钟，以前全手工要几小时；封面工厂已经上线，17 个模板。从几小时到 10 分钟，这个数我自己量的。

（视觉占位：闭环图 —— 预测 → 发布 → T+3 抓数 → 对账 → 写回 成环；配 CoverLab 截图 + 「56 秒 / 10 分钟」大字）

## Page 5 · 商业模式 + Roadmap

**页标题：agency-in-a-box —— 从口播到所有内容形态**

要点：
- 三段走：①现在——自用验证，跑通流程和数据；②近期——订阅制「盒装 agency」，卖给口播/知识类创作者；③远期——校准数据成迁移壁垒
- 切入点：知识类口播创作者（beachhead），最痛、最愿付费
- Roadmap 扩展到教程、vlog、播客切条 —— 换形态只是换套提示词模板
- 技术底座：Claude 驱动，复用两个开源项目（OpenMontage 素材检索 / social-auto-upload 半自动发布），不重复造轮子

讲话稿：
> 商业分三段。现在自用验证。近期做成订阅制，卖给口播和知识类创作者——相当于给每人配家盒装 MCN。远期，校准数据就是护城河，用得越久越离不开。第一刀切知识类口播创作者，最痛也最愿付钱。之后往教程、vlog、播客切条扩，因为换形态只是换套提示词模板。技术上 Claude 驱动，复用了两个开源项目，一个做素材检索一个做半自动发布，不重复造轮子。

## Page 6 · 团队 + 结尾金句

**页标题：一个人 + 一群 Agent —— 人做判断，AI 干活**

要点：
- 团队：刘涵一个人，非技术背景，加一群 Agent。这产品今天用 Claude 现场搭的
- 一个人靠 Agent 跑完一整家内容公司的活 —— 正是这场黑客松要证明的一人公司。我不是讲愿景，我本人就是那个 OPC
- 每个工位 Agent 干活、人审核放行，发布按钮永远人点。不是 AI 取代创作者，是一个创作者顶一整个团队
- Creator OS —— 你的一人媒体公司

讲话稿：
> 团队就我一个人，非技术背景，加一群 Agent，这产品今天用 Claude 现场搭的。一个人靠 Agent 跑完一整家内容公司的活——这不正是黑客松想证明的一人公司吗？我不是讲愿景，我本人就是那个 OPC。我们的原则是：人做判断，AI 干活。每个工位 Agent 干、人审核放行，发布按钮永远人点。不是让 AI 取代创作者，是让一个创作者顶一整个团队。Creator OS，你的一人媒体公司。谢谢。
